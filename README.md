# ✈️ Cercador de Vols (API Backend)

Aquest projecte és un simulador de backend per a un comparador de vols. Està dissenyat per gestionar i consultar rutes aèries, aeroports, companyies i capacitats dels avions, oferint una estructura sòlida i escalable orientada a APIs.

## 🛠️ Tecnologies

* **Framework:** Laravel (PHP)
* **Base de Dades:** SQLite (per a desenvolupament local)
* **Arquitectura:** MVC orientat a Serveis i API REST
